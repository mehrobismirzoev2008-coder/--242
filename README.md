## 📖 Описание проекта

Random Quote Generator — это графическое приложение на Python с использованием библиотеки tkinter.

### Основные возможности:

- 🎲 **Генерация случайных цитат** — одна кнопка и новая цитата каждый раз
- ➕ **Добавление своих цитат** — пополните коллекцию любимыми высказываниями
- 🔍 **Фильтрация по автору и теме** — находите нужные цитаты за секунды
- 📜 **История всех цитат** — сохраняются все сгенерированные цитаты с датой и временем
- 💾 **Автосохранение в JSON** — данные не теряются после закрытия программы
- ✅ **Проверка ввода** — защита от пустых полей при добавлении цитат
